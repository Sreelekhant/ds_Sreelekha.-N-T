# Data Science Assignment – Web3 Trading

## Project Overview
This project analyzes how trader behavior (profit, leverage, trade size) changes depending on market sentiment (Fear vs Greed).

## Folder Structure
- notebook_1.ipynb → Google Colab main notebook  
- csv_files/ → All CSV data files  
- outputs/ → All PNG images  
- ds_report.pdf → Final insights report  

## How to Run
1. Open notebook_1.ipynb in Google Colab  
2. Run all cells  
3. Processed CSVs & charts will be saved automatically
